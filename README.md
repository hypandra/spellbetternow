# SpellBetterNow

SpellBetterNow is an adaptive spelling practice app for kids. Learners complete short mini-sets of words, get immediate feedback, and build confidence over time.

## Features

- Word banks with adjustable difficulty
- Guided spelling sessions in mini-sets
- Progress tracking and mastery insights
- Text-to-speech prompts for words, definitions, and examples

## Tech Stack

- Next.js 16 (App Router) + React 19
- BetterAuth for authentication
- Supabase PostgreSQL for storage
- OpenAI TTS for speech
- Optional Redis caching (Upstash or Vercel KV)
